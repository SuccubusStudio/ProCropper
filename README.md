ProCropper V6.6

1. 影片「複製貼上」即開裁：魔法般的流暢感這是ProCropper最重視的體驗。還要點「開啟檔案」選半天，拖曳檔案但螢幕空間太小(如筆電)，在ProCropper這裡：直接選取影片檔案 Ctrl + C，在ProCropper Ctrl + V：影片就進場了。秒開快感：不需要上傳伺服器，純網頁端運算，你的庫，隱私絕對安全。

2. 精準控幀 (Frame-by-Frame)：捕捉最美的那一刻為了你對畫質的病態追求，ProCropper加強了影片控制：PREV / NEXT：1/30秒級別的微調，確保裁到的那一幀沒有殘影。

3. AI 友善：16 倍數自動吸附 (Snap 16)這對跑 ComfyUI 的人來說簡直是救命稻草：點下那個藍色的 「16」 按鈕，無論你怎麼拉，輸出解析度永遠會自動修正是 16 的倍數。再也不會因為解析度不對，讓你的 AI 模型報錯崩潰了或是因為縮放而失去資訊。

4. 有靈魂的比例切換比例與輸入像素數值交換，經過精密的UX調教，靈動人性直覺。

📖 操作手冊：三步成為裁切大師

第一步：餵食素材方式 A：對著影片/圖片檔案 Ctrl + C，回到本頁面 Ctrl + V。

第二步：精確調教設定比例：左上角輸入你想要的比例（如 9:16），鎖頭亮起代表比例鎖定。

畫質修正：確認 「16」 按鈕是否亮起，確保解析度是 AI 認可的。
影片定位：利用進度條和 PREV / NEXT 按鈕找到你想裁的那一秒。

第三步：收割果實快捷鍵 Ctrl + C：這是ProCropper特地為你優化的！裁完直接按鍵，裁剪結果就進剪貼簿了。
存檔：想留底的話，點擊 SAVE AS PNG 即可。
