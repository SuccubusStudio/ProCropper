🎨 1. 視覺與 UI 特色

這部分決定了你工作時的爽感，目前這版走的是賽博深邃風：

動態環境背光（Ambient Background）：背景會根據你上傳的圖片內容產生模糊的動態光暈，視覺層次感很強。

動態網格背景（Mesh Flow）：背景有深色的流動效果，很有高級感。

毛玻璃工具欄：導航欄和頁尾使用了高強度模糊與飽和度提升，確保 UI 在任何圖片背景下都清晰可見。

Toast 通知系統：執行複製、儲存或填滿操作時，上方會有簡短的狀態提醒。



🛠️ 2. 影像裁切與轉換功能

這部分是你最核心的操作邏輯：

多比例切換：預設了 1:1, 4:3, 16:9, 21:9, 9:16 等多種專業比例。

鎖定機制（Lock Ratio）：可以自由切換「固定比例」或「自由裁切」模式。

像素精準度（Pixels Input）：可以直接手動輸入 W 和 H 的像素值來調整裁切框。

16 像素對齊（Snap to 16px Grid）：這是精華！ 專門為影片編碼和 AI 擴散模型設計，確保輸出的長寬都是 16 的倍數。

一鍵填滿（Fill Btn）：快速將裁切框拉滿到整個原始圖片範圍。

變換工具：支援逆時針 90 度旋轉以及水平鏡像翻轉。



🎞️ 3. 強大的影片幀處理

這讓你的工具直接從「修圖」升級到「影片處理」：

自動抽幀系統：上傳影片後，會根據設定的 FPS 自動將影片拆解成一張張圖片。

自定義 FPS：預設 16 FPS，方便控制抽幀的密度。

影片導航 UI：包含時間軸拉條、正負 1 幀、正負 10 幀的快速切換。

加載遮罩（Loading Overlay）：在處理大型影片檔案時，會有進度條提示，避免網頁看起來像當機。



⌨️ 4. 工作流效率功能

這些是為了讓你用起來「手感更好」的細節：

三合一上傳：支援直接 Ctrl+V 貼上、拖曳進場、或是雙擊上傳。

全域鍵盤快捷鍵：

Ctrl + C：快速執行裁切並複製到剪貼簿（會自動判斷是否在輸入框內，避免衝突）。

方向鍵（左右）：在影片模式下快速切換前後幀。

剪貼簿整合：直接將裁切後的結果轉為 Blob 格式寫入剪貼簿，方便你直接貼進 Eagle 或 ComfyUI。
